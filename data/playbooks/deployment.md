# Deployment

Skills: Deployment, APIs, Cloud Serving

Objective:
Learn how to deploy and serve GenAI applications reliably in real-world environments.

What to learn:
- Streamlit vs FastAPI for serving ML apps
- Environment variables and secrets management
- Containerization basics
- Deployment constraints and scaling considerations

How to prove:
- Deploy a Streamlit or FastAPI GenAI app
- Secure API keys using environment variables
- Demonstrate local and cloud deployment

Tools:
- Streamlit
- FastAPI
- Docker
