# GenAI Engineer Requirements
Skills: Python, Git, Docker, FastAPI, Streamlit, LLM, RAG, Embeddings, Vector Database, LangChain, LlamaIndex, CI/CD, Cloud, MLOps, Transformers, Deployment

A GenAI Engineer typically builds LLM-powered applications with retrieval, evaluation, and production deployment.
Key responsibilities:
- Build RAG pipelines and choose chunking / retrieval strategies
- Integrate vector databases (Chroma, FAISS, Qdrant, Pinecone)
- Deploy apps (FastAPI/Streamlit), manage secrets, latency, monitoring
- Add evaluation/guardrails to reduce hallucinations
