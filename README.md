# CV Skill Gap Analyzer (RAG + Vector DB)
---
[Try -> APP](https://skill-gap-analyzer-ai.streamlit.app/)

Upload a CV (PDF/DOCX), choose a target role (or enter a custom role), optionally paste a job description, and get:
- skill extraction with evidence scoring
- skill gap analysis vs role requirements
- a personalized weekly roadmap

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
