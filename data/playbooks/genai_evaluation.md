# GenAI Evaluation

Skills: Evaluation, Monitoring, Hallucination Detection

Objective:
Learn how to evaluate and monitor GenAI systems for correctness, reliability, and grounding.

What to learn:
- RAG evaluation metrics (groundedness, recall)
- Hallucination detection techniques
- Offline vs online evaluation
- Human-in-the-loop feedback

How to prove:
- Measure retrieval hit rate
- Evaluate answers against source documents
- Track failure cases and improvements

Tools:
- Custom evaluation scripts
- LangChain evaluation utilities
- Logging and monitoring tools
