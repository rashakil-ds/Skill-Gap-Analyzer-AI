# RAG Playbook

Skills: RAG, Embeddings, Vector Database

Objective:
Learn how to design, evaluate, and deploy a retrieval-augmented generation system.

What to learn:
- Chunking strategies, top-k retrieval, metadata filters
- Embeddings and similarity search
- Prompting with citations and grounded answers

How to prove:
- Build a document Q&A app with citations
- Add reranking or query rewriting
- Add evaluation: groundedness, retrieval hit rate

Tools:
- Chroma / FAISS
- Sentence-Transformers
- LangChain (retrieval utilities)
