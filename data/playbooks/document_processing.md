# Document Processing

Skills: PDF Parsing, Text Cleaning, Chunking

Objective:
Learn how to prepare raw documents for downstream retrieval and LLM consumption.

What to learn:
- PDF and document parsing techniques
- Text cleaning and normalization
- Chunking strategies for long documents
- Metadata design (source, role, section)

How to prove:
- Ingest PDFs and markdown files into a vector DB
- Compare retrieval quality with different chunk sizes
- Demonstrate improved retrieval with metadata

Tools:
- pdfplumber
- LangChain text splitters
- Python text processing libraries
